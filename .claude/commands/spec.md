# /spec

Use:

- `skills/solo-kanban-core/SKILL.md`
- `skills/solo-kanban-planning/SKILL.md`

## Goal

Create a real spec for the active task.

## Do

1. Require `requirements.md`.
2. Use `research.md` when present.
3. Create `tasks/<TASK-ID>/Spec.md`.
4. Define problem, goals, non-goals, key decisions, scope, acceptance criteria, and risks.
5. Keep the spec narrow enough for a mergeable slice.
