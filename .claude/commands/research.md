# /research

Use:

- `skills/solo-kanban-core/SKILL.md`
- `skills/solo-kanban-planning/SKILL.md`

## Goal

Create the task research artifact before spec or implementation.

## Do

1. Read the current task `requirements.md`.
2. Inspect relevant code, docs, config, and planning artifacts.
3. Write `tasks/<TASK-ID>/research.md`.
4. Capture findings, options, recommendation, risks, and the implication for `/spec`.
5. State whether research changed the likely scope.
