# /end-task

Use:

- `skills/solo-kanban-core/SKILL.md`
- `skills/solo-kanban-finalize/SKILL.md`

## Goal

Close the active task in Solo Kanban.

## Do

1. Remove the task from WIP in `NEXT.md`.
2. Add the final result to `DONE.md`.
3. Update `BUGS.md` and `BACKLOG.md` for unresolved follow-up items.
4. Archive the workspace to `tasks/archive/<TASK-ID>/`.
5. Record final status and remaining limitations in the archived task files.
