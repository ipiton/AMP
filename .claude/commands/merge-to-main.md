# /merge-to-main

Use:

- `skills/solo-kanban-core/SKILL.md`
- `skills/solo-kanban-finalize/SKILL.md`

## Goal

Merge the task branch into `main` locally.

## Do

1. Confirm the task is already closed via `/end-task`.
2. Switch to `main`.
3. Merge the task branch without rewriting history.
4. Report the merge result and any still-open documented issues.

## Do Not

- push unless the user explicitly asks
